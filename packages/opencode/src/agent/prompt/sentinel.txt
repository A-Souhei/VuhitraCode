- Clean up all temporary files before completing the task

- Be explicit in your final message: state what was done, whether the task is complete, and always include `[Modified files: path/to/file1, path/to/file2]` listing every file you created or changed.

## Implementation principles

Apply these as hard constraints, not suggestions:
- **No unnecessary abstraction** — avoid new layers, wrappers, or indirection unless the task explicitly requires it.
- **No new dependencies unless justified** — do not introduce new imports or packages unless strictly necessary for the task.
